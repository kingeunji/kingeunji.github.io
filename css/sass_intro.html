    # SASS란?

    Sass(Syntactically Awesome Style Sheets) 는 CSS pre-processor 로서, 복잡한 작업을 쉽게 할 수 있게 해주고, 코드의 재활용성을 높여줄 뿐 만 아니라, 코드의 가독성을 높여주어 유지보수를 쉽게 해준다.

    # 1. 변수기능 도입

    Sass는 CSS에 변수 기능을 도입해준다.

    $primary-color: #333;

body {
  background-color: $primary-color;
}

    # 2. 수학 연산자 사용가능

    기존 CSS에서 calc() 함수를 사용했던 연산이 가능해졌다!
    단, 단위를 통일해서 사용해줘야 함!

    .container { width: 100%; }


article[role="main"] {
  float: left;
  width: 600px / 960px * 100%;
}

aside[role="complementary"] {
  float: right;
  width: 300px / 960px * 100%;
}

    기존 CSS 에선 특정 선택자 안에 선택자를 스타일링 하려면 위 코드처럼 써줘야된다.
    프로젝트가 커지면 유지보수가 어렵다는 단점이 있다.

    반면 Sass에서는 부모 선택자를 리퍼런스 할떄는 & 문자를 사용하면서 간단하게 작성할 수 있다.

    /* Sass */
.container {
    width: 100%;
    h1 {
        color: red;
        &:hover {
          color: yellow;
        }
    }

}

    # 5. import

    import 기능은 스타일들을 여러 파일들로 나누고, 다른 파일에서 불러와서 사용하는 기능이다.

    @import "layout";

    # 6. 상속

    Sass 에서 특정 선택자를 상속 할 때, @extend directive를 사용한다.

    .box {
  border: 1px solid gray;
  padding: 10px;
  display: inline-block;
}

.success-box {
  @extend .box;
  border: 1px solid green;
}

    # 7. Mixin (믹스인)

    Mixin는 extend 와 비슷하지만 argument (인수) 를 받을 수 있다.
    mixin 을 선언 할 떄는 @mixin directive 를 사용하며, 이를 사용 할 때는 @include directive 를 사용한다.

    @mixin headline ($color, $size) {
  color: $color;
  font-size: $size;
}

h1 {
  @include headline(green, 12px);
}
